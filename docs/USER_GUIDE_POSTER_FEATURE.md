## Troubleshooting

### Poster doesn't download
- Check browser download settings
- Try a different browser
- Ensure sufficient storage space
- Check browser console for errors

### QR code won't scan
- Ensure printed quality is clear
- Check QR code isn't damaged
- Try different smartphone camera apps
- Verify internet connection

### Poor print quality
- Check printer DPI settings
- Use higher quality paper
- Ensure printer has color cartridge
- Try printing at higher quality settings

### Credit issues
- Free tier users need 45 credits (25 poster + 20 image)
- Check credit balance before generating
- Upgrade to Pro for unlimited posters
